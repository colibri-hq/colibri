Welcome to the first Community Spotlight of 2025! Each month, we'll highlight amazing contributions and projects from the Colibri community.

## Featured Contributions

### Calibre Import Tool

Community member **@bookworm42** created a fantastic tool for migrating Calibre libraries to Colibri. The tool preserves:

- Reading lists and custom columns
- Cover images and metadata
- Tags and categories

[Check it out on GitHub](https://github.com/example/calibre-to-colibri)

### Improved Search

**@searchmaster** contributed significant improvements to our full-text search:

- Fuzzy matching for typos
- Author name variations
- Series number parsing

### Accessibility Fixes

Thanks to **@a11y-champion** for their work on accessibility:

- Screen reader improvements
- Keyboard navigation fixes
- High contrast mode support

## Community Stats

This month by the numbers:

- **23** pull requests merged
- **47** issues closed
- **156** new stars on GitHub
- **12** new contributors

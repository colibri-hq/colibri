We're thrilled to announce **Colibri v3**, the most significant update to our self-hosted ebook library platform since its inception.

## What's New in v3

### Metadata Enrichment

One of the most requested features has finally arrived. Colibri can now automatically enrich your ebook metadata by querying multiple authoritative sources:

- **Open Library** - Access millions of book records
- **WikiData** - Rich structured data about authors and publishers
- **ISNI** - International Standard Name Identifier lookups
- **VIAF** - Virtual International Authority File integration

### Improved Collections

Collections have been completely redesigned:

- Drag-and-drop organization
- Custom collection icons with emoji picker
- Nested subcollections support
- Smart collections based on tags and metadata

### New Documentation Site

You're reading it! Our new documentation site features:

- Full-text search powered by Pagefind
- Dark mode support
- Improved navigation and organization
- This development blog you're reading now
